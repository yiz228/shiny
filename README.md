# WIC Toddler Overweight Prevalence Explorer

This repository contains a Shiny App created for VTPEH 6270 Check Point 07. The app allows users to explore racial and ethnic differences in overweight prevalence among children participating in the Women, Infants, and Children (WIC) program.

## App Goal

The goal of this app is to provide an interactive visualization of overweight prevalence among WIC toddlers by race/ethnicity and year. Users can select a year, choose a plot type, and select race/ethnicity groups to compare prevalence patterns.

## Data Source

The data used in this app come from CDC Data.CDC.gov, specifically the *Nutrition, Physical Activity, and Obesity - Women, Infant, and Child* dataset.

## Files

- `app.R`: Shiny App code
- `Nut_Data.csv`: Dataset used in the app

## How to Use the App

Users can interact with the sidebar controls to:

- Select a year
- Choose between a boxplot and a bar chart
- Select specific race/ethnicity groups

The data visualization and summary text update based on the selected options.
